## 2. `근태 ( 3월 )` 시트 좌표 매핑

### 헤더 영역 (Row 1~6)
- **`O1`** = 연도 (예: 2026)
- **`Q1`** = 월 (예: 3)
- `S2` = 시작일 (=DATE 수식)
- `U2` = 종료일 (=EDATE 수식)
- Row 5: 컬럼 헤더 (`번호`/`성명·성별`/`일수`/`구분`/날짜...)
- Row 6: 요일 표시

### 데이터 시작
- **`Row 7`** 부터 직원 데이터 시작 (Row 1~6은 헤더)

### 직원 블록 (6행 구조, 직원당)

| 오프셋 | 행 | F열 라벨 |
|---|---|---|
| +0 | basic_row | **`기본`** |
| +1 | basic_row+1 | **`연장`** |
| +2 | basic_row+2 | **`심야`** |
| +3 | basic_row+3 | **`특근`** |
| +4 | basic_row+4 | **`특잔`** |
| +5 | basic_row+5 | **`지각 조퇴`** (공백 있음) |

### 컬럼 매핑

| 컬럼 | 의미 |
|---|---|
| A | 직원명 참조 (수식: `=D7`) |
| B | 직원명+라벨 (수식: `=$D7&F7`) |
| C | 번호 |
| **D** | **직원명** (예: `이희영`) |
| E | 일수 합계 (수식: `=COUNT(H7:AL7)`) |
| **F** | **행 라벨** (`기본`/`연장`/...) |
| G | 시급 |
| **H** | **3/1 (1일)** |
| I | 3/2 (2일) |
| ... | ... |
| **AL** | **3/31 (31일)** |
| **AM** | **월말 주휴 보정칸** (헤더 `AM5="주휴"`) |

```python
# Python (1-based)
DAY_TO_COL = {day: 7 + day for day in range(1, 32)}
# day=1 → col=8 (H), day=31 → col=38 (AL)
MONTHLY_ADJUSTMENT_COL = 39  # AM
```

|---|---|
| 기본 행 | `8` |
| 연장 행 | **`2`** |

### 3.3 일요일 주휴 (자동)

조건 충족 시 (월~금 인정 + 다음주 월 인정):

| 위치 | 값 |
|---|---|
| 기본 행 | `8` |
| 연장 행 | **`"주휴"`** |

### 3.4 평일 공휴일 유급 (예: 3/2 삼일절 대체)

| 위치 | 값 |
|---|---|
| 기본 행 | `8` |
| 연장 행 | **`"유급"`** |

### 3.5 연차/반차/반반차 (잔업 없음)

PDF: `연차` 또는 `반차` 또는 `반반` (단독)

| 위치 | 값 |
|---|---|
| 기본 행 | `8` |
| 연장 행 | **`"연차"`** / **`"반차"`** / **`"반반차"`** |

### 3.6 ⚡ 휴가 + 잔업 동시 발생 (이채림 3/20 패턴)

PDF: `반차  20:00  2.0`

> **완성 폼 확인**: `이채림` row=127, 3/20 (col=27)
> - F127 `기본` → AA127 = `8`
> - F128 `연장` → AA128 = `2` (잔업 숫자)
> - **F129 `심야` → AA129 = `"반차"`** ← 3번째 행에 휴가 텍스트
> - F130~132 → 빈칸

| 위치 | 값 |
|---|---|
| 기본 행 (basic) | `8` |
| 연장 행 (basic+1) | **`2`** (잔업 숫자) |
| **심야 행 (basic+2)** | **`"반차"`** / **`"연차"`** / **`"반반차"`** |
| 특근/특잔/지각조퇴 | 빈칸 |

### 3.7 토요일/일요일 근무

PDF: 주말에 `08:30  17:30  8.0` 같이 시간 + 잔업 숫자

| 위치 | 값 |
|---|---|
| 기본 행 | 빈칸 |
| **연장 행** | **숫자** (예: `8`) |
| 특근 행 | 빈칸 (특근 행에 X — 연장 행에) |

### 3.8 지각/조퇴 (잔업 명시 없는 비정상 시간)

PDF: `13:30  20:00` (잔업 숫자 없음) → 실근무 부족분

| 위치 | 값 |
|---|---|
| 기본 행 | `8` |
| 지각 조퇴 행 (basic+5) | **`-2`** (실근무 6시간이면 8-6=2 부족) |
| 연장 행 | 빈칸 (자동 잔업 계산 X) |

부족분 = 8 - (end_hour - start_hour - 1점심)

### 4.3 보조 항목 자동 제외

`조윤서B`, `조윤서_퇴직금`, `조윤서정산` 등은 PDF 매칭 대상에서 제외:

```python
def is_auxiliary_employee(name):
    n = normalize_name(name)
    # 한글 + 영문 대문자 1글자 (조윤서B)
    if len(n) >= 2 and n[-1].isascii() and n[-1].isalpha() and n[-1].isupper():
        if '가' <= n[-2] <= '힣':
            return True
    # 한글 접미사
    for suffix in ("퇴직금", "정산", "별도", "예비"):
        if n.endswith(suffix) and len(n) > len(suffix):
            return True
    return False
```
