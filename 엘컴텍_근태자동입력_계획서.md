### 1.3 근태 시트 구조 (직원당 6행)

`기본 / 연장 / 심야 / 특근 / 특잔 / 지각조퇴`
- 일자 컬럼: H열(1일) ~ (월말). 헤더 5행이 날짜.

### 1.4 PDF → 근태행 매핑 규칙 (검증 완료)

**평일 (구분=평일)**
- 평일기본 → `기본`행 (정상 8, 지각 시 7 등)
- 평일연장 → `연장`행 (예 2:30 → 2.5)
- 평일야간 → `심야`행
- 결근 → `기본`행 공란(수동: 휴무/연차/반차 판단 필요) → **검토 플래그**

**토요일 (구분=무휴, 주간-주말)**
- 근무 시 휴일기본 → `연장`행 = 8  (토요일 근무 = 연장행)
- 휴일연장 → `특잔`행

**일요일 (구분=주휴)**
- 물리적 근무(휴일기본>0) → `특근`행 = 8 (일요일 근무 = 특근행)

**주휴수당 (개근룰 — 검증 완료)**
- 각 일요일에 대해 직전 월~토 주가 **개근**(결근 0)이면:
  - `기본`행[일] = 8
  - `연장`행[일] = "주휴"(텍스트)
- 연차는 개근 깨지 않음(법정). 결근만 개근 깨짐.
- 검증: 박미선 = 4주 전부 개근 → 5·12·19·26 전부 주휴 / 김춘희 = 4/15·16 결근 → 4/19 주휴 없음 ✓

**지각조퇴 (지각조퇴행, 음수)** — 단순 비례 아님(0:55→-1, -4/-2/-3 사례). 자동 best-effort + **검토 플래그**(수동 보정 여지).

### 1.5 자동/수동 경계
- 자동: 기본·연장·심야·특근·특잔 + 주휴 개근마커
- 수동(검토 플래그): 결근→연차/반차/휴무 재분류, 지각조퇴 deduction 미세조정
